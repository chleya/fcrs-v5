# EXPERIMENT 4: 截断机制深度验证

## 1. 理论背景

### 核心发现

```
智能 = 截断 + 回流
    ↓
能量截断 → 硬预算
信息截断 → top-k竞争
    ↓
边界出现 → 自我 → 智能
```

### 河的比喻

```
顺放（无截断）     横放（截断）
水流→→→→→      水流→←←←←←
                    ||||||||
                      ↓↓↓
                    水花飞溅
                    结构涌现
```

---

## 2. 研究问题

### Q1: 截断强度 vs 结构稳定性

更强的截断 → 更稳定的结构？

### Q2: 截断类型对比

| 类型 | 机制 |
|------|------|
| 容量截断 | 硬预算 |
| 信息截断 | top-k |
| 能量截断 | 阈值门限 |

### Q3: 回流的作用

有回流 vs 无回流 → 结构能否维持？

---

## 3. 实验设计

### 3.1 截断强度实验

**变量**: Budget B ∈ {8, 12, 16, 24, 32}

**控制**: k=2 (固定竞争)

**预期**:
```
维度
  ↑
32 |           ●
24 |        ●
16 |     ●
12 |  ●
8  |●
  +-------------> Budget
```

**预言**: Budget越小 → 维度越稳定

---

### 3.2 竞争强度实验

**变量**: k ∈ {1, 2, 4, 8, 16}

**控制**: Budget=16 (固定容量)

**预期**:
```
维度
  ↑
k=1  |●
k=2  |  ●
k=4  |    ●
k=8  |       ●
k=16 |          ●
  +-----------------> k
```

**预言**: k越小 → 维度越低（更稀疏）

---

### 3.3 截断类型对比

| 系统 | 截断类型 | 实现 |
|------|----------|------|
| S1 | 无 | 无限制 |
| S2 | 容量 | max_dims=B |
| S3 | 信息 | top-k |
| S4 | 能量 | 阈值门限 |
| S5 | 组合 | 容量+信息+能量 |

---

### 3.4 回流实验

**变量**: 回流类型

| 系统 | 回流 | 说明 |
|------|------|------|
| R0 | 无 | 无反馈 |
| R1 | 梯度 | 误差反传 |
| R2 | 注意力 | 重要信息强化 |
| R3 | 预测 | 期望vs现实 |

**预期**:
```
稳定性
  ↑
R3 |       ●●●●●●
R2 |    ●●●●
R1 |  ●●
R0 |●
  +-----------------> 回流强度
```

**预言**: 有回流 → 更稳定

## 5. 关键指标

### 5.1 截断指标

| 指标 | 公式 |
|------|------|
| 截断效率 | active_dims / max_dims |
| 截断强度 | 1 - (active_dims / max_dims) |

### 5.2 回流指标

| 指标 | 公式 |
|------|------|
| 反馈强度 | correlation(prediction, actual) |

| 能量效率 | performance / dimension |

## 6. 预期结果

### 6.1 截断强度

- Budget越小 → 维度越稳定
- 存在临界Budget: 低于它结构崩溃

### 6.2 竞争强度

- k越小 → 维度越低
- 存在最优k: 太低性能下降

### 6.3 截断类型

- 组合截断 > 单类型截断
- 能量截断最敏感

### 6.4 回流

- 预测回流最稳定
- 无回流维度持续漂移

---

## 7. 验证标准

### 7.1 假设H4: 截断强度与稳定性正相关

- 验证: Budget与维度波动性负相关

### 7.2 假设H5: 组合截断最优

- 验证: S5性能 > S1-S4

### 7.3 假设H6: 回流维持结构

- 验证: 有回流组维度变化 < 无回流组

---

## 8. 失败标准

- 截断强度与稳定性无关
- 组合不优于单类型
- 回流无显著作用

---

## 9. 技术实现

### 9.1 截断类型实现

```python
# 容量截断
if current_dims >= max_budget:
    return False  # 禁止扩张

# 信息截断  
top_k_indices = argsort(activation)[-k:]
mask = zeros_like(activation)
mask[top_k_indices] = 1

# 能量截断
output = threshold * (activation > threshold)
```

### 9.2 回流实现

```python
# 梯度回流
grad = backward(loss)

# 注意力回流  
attention = softmax(activation)
activation = attention * activation

# 预测回流
prediction = forward(state)
error = target - prediction
state = state + learning_rate * error
```
