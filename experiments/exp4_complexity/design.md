# Experiment 4: Task Complexity × Capacity

**Date**: 2026-03-08  
**Question**: Does task complexity drive representation capacity under structural capacity cost?

---

## 1. 实验设计

固定 λ = 0.1 (regulated regime)

变量: task complexity ∈ {3, 5, 10, 20, 50}

---

## 2. 理论预测

| 结果 | 含义 |
|------|------|
| complexity ↑ → dimension ↑ | 自适应容量分配 |
| dimension ≈ constant | 容量由λ主导 |
| complexity ↑ → loss ↑ | 系统无法适应 |

---
