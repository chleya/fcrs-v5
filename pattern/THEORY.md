# 智能的源头：规律检测

**日期**: 2026-03-08

---

## 核心理论

```
规律(源头) + 接收结构 = 智能
```

### 核心假设

> **智能 = 检测规律 + 响应规律的能力**

---

## 关键概念

| 概念 | 定义 |
|------|------|
| **规律** | 环境中的可预测结构 |
| **检测** | 系统识别规律的能力 |
| **响应** | 系统根据规律调整行为 |
| **接收结构** | 能响应规律的任何结构 |

---

## 与现有理论的连接

| 理论 | 对应 |
|------|------|
| 信息论 | 规律 = 减少熵 = 信息 |
| 柯尔莫格罗夫复杂度 | 智能 = 压缩规律 |
| 预测加工 | 大脑 = 预测规律的机器 |
| FEP | 最小化惊奇 = 找到规律 |
