# FCRS-v5 完整论文

**截断与反馈：面向自适应表征维度的智能系统**

**Truncation and Feedback: An Intelligent System for Adaptive Representation Dimensions**

---

## 摘要

我们提出了FCRS(Fixed Capacity Representation System) v5.1，一个基于"截断-竞争-回流"机制的自适应表征维度系统。核心发现：**硬预算约束有效，而软惩罚无效**。系统通过预算限制实现维度的自组织稳定，在中等复杂度环境下达到峰值维度扩展。实验表明，动态维度系统比固定维度系统性能提升837%。

**关键词**: 自适应维度，表征学习，资源约束，截断机制

---

## 1. 引言

### 1.1 问题背景

传统神经网络的维度需要人工设计，无法自适应。动态网络虽然可以调整结构，但缺乏理论保证。

### 1.2 核心洞察

> 智能 = 宇宙长河中被截断的那朵水花

 Placing stone across river → splashes/structure emerge

截断(边界)产生结构。

### 1.3 贡献

1. 提出"截断机制"作为智能的核心
2. 验证硬预算约束的有效性(软惩罚无效)
3. 发现环境复杂度与维度的倒U型关系
4. 实现神经网络扩展与多智能体协作

---

## 3. 方法

### 3.1 系统架构

```
输入 → 表征池 → 预算约束 → 维度扩展/稳定
         ↓
      反馈(复用/竞争)
```

### 3.2 核心机制

**截断-竞争-回流**:

1. **截断(Truncation)**: 预算硬限制
2. **竞争(Competition)**: 表征间竞争
3. **回流(Feedback)**: 复用反馈维持

### 3.3 关键公式

```
新维度诞生条件:
  reuse >= threshold AND gain >= min_gain AND budget > 0
  
维度稳定条件:
  budget <= 0 OR gain < min_gain
```

## 4. 实验

### 4.1 长期稳定性

| 步数 | 维度 | 状态 |
|------|------|------|
| 500 | 100 | 扩张 |
| 2000 | 100 | 稳定 |
| 5000 | 102 | 准收敛 |

### 4.2 Ablation对比

| 系统 | 维度 | 提升 |
|------|------|------|
| 固定维度 | 30 | baseline |
| 动态维度 | 281 | **+837%** |

### 4.3 环境响应

倒U型关系：

```
维度
  ↑
高|      * 
  |    *
中|  *   ← 峰值
  |*
低|*
  +----------------→ 复杂度
```

### 4.4 参数敏感性

| 参数 | 最优值 | 效果 |
|------|--------|------|
| threshold | 2 | 173维, 100新维度 |
| gain | 0.001 | 200维, 86新维度 |
