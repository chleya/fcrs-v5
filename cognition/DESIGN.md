## 问题2: 如何确定"长出认知"？

```
认知标志 = 可测量的行为
```

| 标志 | 测试 | 成功条件 |
|------|------|---------|
| 模式识别 | 周期性输入 | 预测误差下降 |
| 泛化 | 新输入 | 误差 < 阈值 |
| 能动性 | 自主寻找 | 主动探索 |
| 记忆 | 保留学到的 | 权重稳定 |

## 实验设计

### Phase A: 最小结构

```
单元数=10, 无连接, 无演化
→ 应该无认知
```

### Phase B: 复杂度

```
单元数=100, 有连接, 无演化
→ 可能有简单学习
```

### Phase C: 完整

```
单元数=100, 有连接, 有演化, 能量约束
→ 应该有认知涌现
```
