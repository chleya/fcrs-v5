# FCRS-v5 理论形式化

## 符号定义

| 符号 | 含义 |
|------|------|
| $x_t$ | t时刻的输入 |
| $R_t$ | 表征池 |
| $r_i$ | 第i个表征 |
| $d_i$ | 表征维度 |
| $D$ | 总维度预算 |
| $f_i$ | 表征适应度 |
| $\theta$ | 阈值参数 |

---

## 系统模型

### 环境层

输入分布: $x_t \sim P(x)$

环境复杂度: $C = H(x)$ (熵)

### 表征层

表征池: $R = \{r_1, r_2, ..., r_n\}$

总维度: $D_{total} = \sum_i d_i$

预算约束: $D_{total} \leq B$ (B=预算)

### 进化层

#### 选择

$$r^* = \arg\max_{r \in R} \text{sim}(x_t, r)$$

#### 竞争

$$\Delta f = f_{new} - f_{old}$$

如果 $\Delta f > \theta$: 保留新表征

---

## 假设形式化

### H1: 资源约束收敛

$$\lim_{t \to \infty} D_{total}(t) \leq B$$

### H2: 竞争选择

$$P(\text{保留}) = P(\Delta f > \theta)$$

### H3: 复杂度正相关

$$D_{total} \propto C$$

---

## 收敛性证明(草稿)

### 定理1: 有限维度

如果 $d_i \geq 1$ 且 $B < \infty$, 则系统维度有限。

证明: $D_{total} = \sum d_i \leq n \cdot \max(d_i) < \infty$
