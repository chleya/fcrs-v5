# FCRS-v5 论文大纲

## 标题

**截断与反馈：面向自适应表征维度的智能系统**

(Truncation and Feedback: An Intelligent System for Adaptive Representation Dimensions)

---

## 摘要

我们提出了FCRS(Fixed Capacity Representation System) v5.1，一个基于"截断-竞争-回流"机制的自适应表征维度系统。核心发现：硬预算约束有效，而软惩罚无效。系统通过预算限制实现维度的自组织稳定，在中等复杂度环境下达到峰值维度扩展。

---

## 1. 引言

### 问题
- 传统神经网络维度固定，需要人工设计
- 自适应系统缺乏理论保证

### 贡献
1. 提出"截断机制"作为智能的核心
2. 验证硬预算约束的有效性
3. 发现环境复杂度与维度的倒U型关系

## 3. 方法

### 3.1 核心机制

```
输入 → 表征池 → 预算约束 → 维度扩展/稳定
         ↓
      反馈(复用/竞争)
```

### 3.2 关键公式

```
新维度诞生条件:
  reuse >= threshold AND gain >= min_gain AND budget > 0
```
