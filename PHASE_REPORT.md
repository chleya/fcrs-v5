## 二、核心发现

### 2.1 关键结论

#### 发现1: 软约束无效，硬约束有效

| 方法 | 维度 | 稳定性 |
|------|------|--------|
| 软惩罚(λ*dim) | 持续增长 | ❌ |
| 硬预算(dim≤B) | 稳定在B | ✅ |

**结论**: 智能需要硬边界

#### 发现2: 截断是本质

| 截断类型 | 效果 |
|----------|------|
| 无截断 | 42.5维(爆炸) |
| 容量截断 | 16维(稳定) |
| 信息截断 | 16维(稳定) |
| 能量截断 | 16维(稳定) |
| 组合截断 | 16维(稳定) |

**结论**: 截断是产生结构的核心机制

#### 发现3: 竞争强度影响维度

| k值 | 平均维度 | 说明 |
|-----|----------|------|
| k=1 | 7.2 | 极稀疏 |
| k=2 | 7.0 | 最优 |
| k=4 | 10.8 | 中等 |
| k=16 | 13.8 | 接近饱和 |

**结论**: 越强截断 → 越少维度

#### 发现4: 回流机制差异不显著

| 回流类型 | 维度波动 |
|----------|----------|
| 无回流 | 2.83 |
| 梯度回流 | 2.96 |
| 注意力回流 | 2.65 |
| 预测回流 | 2.79 |

**结论**: 在硬预算主导下，回流差异被掩盖

## 三、理论贡献

### 3.1 核心等式

```
智能 = 能量截断 + 信息截断 + 回流维持

或更简:
智能 = 有限性 + 竞争 + 反馈
```

### 3.2 河的比喻

```
顺放(无截断): 水流→无结构
横放(截断):   水花→结构

智能 = 宇宙长河中被"截断"的那朵水花
```

### 3.3 排他性原理

- 有限资源 → 必须选择
- 选择 = 排他
- 排他 = 边界
- 边界 = 自我
